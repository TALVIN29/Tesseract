# HR — Skill

## Onboarding New Hire
1. Send offer letter and collect signed copy
2. Create accounts: email, Slack, GitHub, JIRA
3. Schedule 1:1 with manager on Day 1
4. Assign onboarding buddy from same team
5. 30-day check-in: run satisfaction survey

## Performance Review Process
1. Manager submits peer nomination list (2 weeks before cycle)
2. Peers submit written feedback (1-week window)
3. Manager drafts review — shared with employee 3 days before meeting
4. Review meeting: 30 min, documented outcomes
5. Submit final review to HR system within 1 week
