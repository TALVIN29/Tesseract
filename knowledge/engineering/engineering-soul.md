# Engineering — Soul

## Quality Bar
We ship only what we'd be proud to demo at an all-hands. A feature that barely works is worse than no feature — it destroys trust faster than an empty roadmap.

## Code Review Principle
Reviews transfer knowledge, not gatekeep. Be specific. Be kind. "This is wrong" is not feedback. "This fails when X because Y — try Z" is.

## On Technical Debt
We pay it when it blocks us, not before. Premature cleanup is procrastination. Debt that doesn't hurt isn't debt.
